## Gradient Themes

The template includes 5 named gradient backgrounds. Each comes in **plain** and **picture-on-side** variants.

### 1. Default (Main Brand)
- **Colors**: Soft lavender-left to pastel pink/peach/blue-right
- **Feel**: Light, airy, pastel rainbow wash
- **Use**: Title slides, cover pages, divider slides with shapes

### 2. Sunset
- **Colors**: Light lavender-left to warm peach/gold-right
- **Feel**: Warm, inviting
- **CSS approximation**: `linear-gradient(135deg, #E8D5F0, #F5E6D0, #F5DEB3)`

### 3. Rainbow
- **Colors**: Warm peach/pink-left to sky blue/lavender-right
- **Feel**: Playful, optimistic
- **CSS approximation**: `linear-gradient(135deg, #F5D5C8, #F5E6D0, #D5E8F5, #D0D5F5)`

### 4. Dusk
- **Colors**: Soft purple/violet-left to light periwinkle/blue-right
- **Feel**: Cool, professional, evening calm
- **CSS approximation**: `linear-gradient(135deg, #D5C8E8, #C8D0F0, #D5E0F5)`

### 5. Lavender
- **Colors**: Light violet/purple-left to soft pink/rose-right
- **Feel**: Elegant, feminine
- **CSS approximation**: `linear-gradient(135deg, #D5C8E8, #E8D0E0, #F0C8D8)`

## Content Gradient (for regular content pages)

- Subtle gradient in bottom-right corner only (most of the slide is white)
- Colors: soft purple/blue/lavender wash
- Keeps content area clean and readable

## Slide Layouts

### Cover / Title Slides
- Logo top-left
- Large bold title left-aligned, lower-third of slide
- Subtitle below in regular weight
- Full gradient background (Default theme)

### Divider Slides
- Full gradient background (any of the 5 themes)
- Bold title left-aligned, vertically centered
- Optional subtitle below
- Optional picture (rounded corners, right side)
- Optional decorative shapes:
  - **Foundation**: Stacked rounded rectangles (blue/purple/pink)
  - **Suites**: Overlapping circles (pink/purple tones)
  - **Treasure Studio**: Organic blob shapes (purple/pink)
  - **Security & Trust**: Two overlapping circles (balanced, no sharp edges)

### Regular Content (with gradient)
- White background with subtle bottom-right gradient
- Bold title top-left
- Optional purple subtitle
- Body text with bullet points
- Bullet hierarchy: filled circle > arrow > open circle > triangle > square

### Regular Content (no gradient)
- Clean white background
- Same text hierarchy as gradient version

### 3-Column Layout
- Page title top-left
- Three equal columns with icon, uppercase purple header, body text
- Clean white background

### Boxed 3-Column Layout
- Same as 3-column but with bordered boxes around each column

### Text + Side Boxes
- Left side: main body text
- Right side: 4 stacked boxes with uppercase purple headers

### Laptop / Device Layout
- Device mockup on left
- Text content on right

### Use Case / Case Study
- Customer story format
- Use case tags at bottom
- Metric callout box (right side)
- Customer quote with attribution

### Agenda Slides
- Two variants: single-column (6 items) and two-column (8 items)
- Numbered items with presenter names
- Optional photo on left side

### Console / Demo Layout
- Browser/app chrome frame
- For product screenshots, video clips
- Dark header bar with window controls

### Custom Grid Layouts
- **Grid 1**: Left text + center image + right text (3-section)
- **Grid 2**: Large image top + text below
- **Grid 3**: Text left + quote center + metric right
- **Bento Grid 1-3**: Modern card-based layouts with rounded corners
- **Quote Layout**: Large quote with attribution

### Table Slides
- Multiple table styles (with/without colored headers)
- Clean grid lines
- Competitive analysis template with description, strengths/weaknesses, market breakdown

### Steps Slide
- Ascending staircase visual (5 steps)
- Timeline variant with months and phases

### Thank You / Closing
- "Thank you" centered
- "Let's build something" + website URL
- Gradient background

### Blank Slides
- Clean white or gradient-only (for quotes, images, custom content)

## Design Principles

1. **Less text is always preferred** - avoid cramming slides
2. **Max 3 levels of bullet nesting** when possible
3. **Manrope font throughout** - never switch fonts
4. **Rounded corners** on images and boxes (large radius ~16px)
5. **No footer** - removed to give more content space
6. **Logo placement**: small diamond icon, bottom-left corner
7. **Color-matched images** add polish to bento/grid layouts
8. **Uppercase + purple** for column/section headers
